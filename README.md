# BOJ Editor

[백준 온라인 저지(BOJ)](https://www.acmicpc.net) 문제 페이지에서 바로 코드를 작성하고 실행할 수 있는 Chrome 확장 프로그램입니다.

문제 페이지를 벗어나지 않고 사이드패널에서 코딩, 테스트, 제출까지 한 번에 할 수 있습니다.

## 주요 기능

- **사이드패널 코드 에디터** — Monaco Editor(VS Code의 에디터 엔진) 기반으로 자동완성, 구문 강조 등을 지원합니다.
- **테스트케이스 실행** — 커스텀 테스트케이스를 추가하고 바로 실행해볼 수 있습니다.
- **코드 자동 저장** — 문제별, 언어별로 작성한 코드가 자동 저장됩니다. 브라우저를 닫아도 코드가 사라지지 않습니다.
- **바로 제출** — 에디터에서 작성한 코드를 백준에 바로 제출할 수 있습니다.
- **다크/라이트 테마** — 취향에 맞게 테마를 전환할 수 있습니다.
- **다양한 언어 지원** — Python 3, C++17, Java 11, C99, Node.js, Kotlin, Go, Rust, Swift, C#, Ruby, D

## 설치

> Chrome 116 이상이 필요합니다.

현재 Chrome 웹스토어에 등록되어 있지 않으며, 최신 버전은 **v0.1.0**입니다.

1. [릴리즈 페이지](https://github.com/kimday0326/boj-editor/releases)에서 최신 버전의 zip 파일을 다운로드합니다.
2. 다운로드한 zip 파일의 압축을 풀어줍니다.
3. Chrome에서 `chrome://extensions`에 접속합니다.
4. 우측 상단의 **개발자 모드**를 활성화합니다.
5. **압축해제된 확장 프로그램을 로드합니다**를 클릭하고 압축을 푼 폴더를 선택합니다.

## 사용 방법

1. [백준](https://www.acmicpc.net)에 로그인합니다.
2. 아무 문제 페이지(예: [1000번](https://www.acmicpc.net/problem/1000))로 이동합니다.
3. 사이드패널에서 에디터가 열리면 언어를 선택하고 코드를 작성합니다.
4. 테스트케이스를 추가한 뒤 **Run**(또는 `Ctrl/Cmd+Enter`)으로 실행 결과를 확인합니다.
5. **Submit** 버튼으로 백준에 제출합니다.
